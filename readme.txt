				AMACHAY - Plataforma de Seguridad Minera 4.0


Descripción
Aplicación web integral diseñada para la **Identificación de Peligros y Evaluación de Riesgos (IPERC)**, análisis cualitativo mediante el modelo **Bow-Tie** y generación automatizada de documentos oficiales (**Anexos 9 y 10**).


Características Principales
* **Módulo IPERC:** Matriz interactiva 5x5 con visualización de puntos críticos.
* **Bow-Tie Dinámico:** Generación de gráficos de barreras preventivas y mitigadoras con exportación a PNG.
* **Generador de PETS/Estándares:** Formatos que cumplen con la normativa peruana, con encabezados y pies de página repetitivos para impresión profesional.
